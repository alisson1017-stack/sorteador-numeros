<h2>Sobre o projeto:</h2>

<p>
Sorteador de números onde o usuário deve informar:
<ul>
<li>a quantidade de números a serem sorteados</li>
<li>o menor número a ser sorteado <i>("Do número")</i></li>
<li>o maior número a ser sorteado <i>("Até número")</i></li>
</ul>

<br>

<s>A cada clique do botão "Sortear", os números são sorteados um a um e adicionados à lista abaixo dos botões até atingir a quantidade de números especificada.</s>
</p>

<p>
  $${\color{green} Versão \space 2.0:}$$
  <br>
  <strong>+A cada clique do botão "Sortear", os números são sorteados e exibidos abaixo dos botões de acordo com os inputs do usuário.</strong>
</p>

<br>

O front-end (HTML e CSS) foi feito pelo time da Alura, já o back-end (JavaScript) foi desenvolvido por mim.
