# Sorteador de Números

![javascript](https://img.shields.io/badge/JavaScript-e0ca28?style=for-the-badge&logo=JavaScript&logoColor=2D2D30)

## Sobre o projeto:

<p>

Sorteador de números onde o usuário deve informar:
<ul>

<li>a quantidade de números a serem sorteados</li>
<li>o menor número a ser sorteado <i>("Do número")</i></li>
<li>o maior número a ser sorteado <i>("Até número")</i></li>

</ul>

A cada clique do botão "Sortear", os números são sorteados um a um e adicionados à lista abaixo dos botões.
</p>

O front-end (HTML e CSS) foi feito pelo time da Alura, já o back-end (JavaScript) foi desenvolvido por mim.

> O durante o desenvolvimento, utilizei Inteligência Artificial somente como ferramenta de pesquisa, a consultando somente quando eu precisava de algum comando que eu ainda não conhecia ou entender melhor o funcionamento de um que eu já estava usando.
